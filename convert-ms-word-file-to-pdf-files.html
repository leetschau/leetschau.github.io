Convert MS Word File to PDF Files

On Ubuntu/Mint, when we receive MS Word documents (.doc or .docx), convert it to pdf, then open it with evince.

sudo apt-get install unoconv
doc2pdf target.docx

doc2pdf save output pdf file into .pdf by default.
