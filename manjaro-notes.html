Manjaro Notes

Search application: pamac search xxx

Install application: pamac install xxx

Update system

First configure the update mirrors: Open Pacman Mirrorlist Generator (WITHOUT proxy), choose China and Generate List. Paste result texts into /etc/pacman.d/mirrorlist.

Check for available updates:

pamac checkupdates -a

Then update system with sudo pamac update, met the following errors:

Loading packages files...
Checking file conflicts...
Error: Failed to commit transaction:
conflicting files:
- dunst: /usr/bin/dunstify already exists in filesystem (owned by dunstify)
- lsb-release: /etc/lsb-release already exists in filesystem (owned by manjaro-release)

Can't find solutions.
