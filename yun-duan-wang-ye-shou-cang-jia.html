云端网页收藏夹

Raindrop.io

raindrop.io 是一个类似于 Pocket 的网络收藏夹， 不用代理速度比加代理快，有 Chrome 和 Firefox 插件。

安装后有两种模式：clipper 和 minApp， 前者只能将当前网页存储到云端，后者则是一个微缩版在线帐号， 可以将当前网页保存进去，也可以在其中搜索收藏的网页。

导入已有数据

Chrome 和 Firefox 的书签管理器（都用 Ctrl-Shift-o 打开） 里可以将书签导出为 HTML 文件，然后导入 raindrop.io， Pocket 在 个人账户（网页右上角头像）的 选项 中也有导出到 HTML 功能，

把这三者都导入到了 raindrop.io 里，后续就用它作为网络收藏夹了。

其他选项

Pocket 的 Chrome 插件点击是保存当前网页，右键第一项是打开收藏列表， 只是速度比 raindrop.io 慢，且需要使用代理。

Eversync 的工作模式与前二者不同， 它将浏览器的书签同步到在线帐号 EverHelper 中（需要使用代理）， 由于两种浏览器书签收藏夹结构不同，同步效果似乎不太好。 从 Chrome 同步到 Firefox 成功了，但 Firefox 到 Chrome 不成功，

在Chrome和Firefox之间同步书签

（2011.12.16 版本）

二者之间用html文件作为中介实现书签同步。

Import / Export bookmarks in Chrome: Wrench Button -> Bookmarks -> Bookmark manager(Ctrl+Shift+O) -> Organize -> Import bookmarks from HTML file/ Export bookmarks to HTML file;

Import / Export bookmarks in Firefox: "Firefox" Button(upper left corner of the browser window) -> Bookmarks -> Show All Bookmarks(Ctrl+Shift+B) -> Import and Backup -> Import bookmarks from HTML file/ Export bookmarks to HTML file;

验证版本：Chrome: 15.0, Firefox: 8.0.1;

这个方法的好处是简单易行，无需使用任何同步工具，不受Chrome和Firefox版本的限制，可以双向同步。
