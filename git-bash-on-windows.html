Git Bash on Windows

Windows 10 的 Git bash 通过 .minttyrc 配置文件可以实现多种效果定义， 例如字体大小、字符集、编码、是否显示滚动条、启动后最大化等，完整内容：

cat ~/.minttyrc
BoldAsFont=-1
FontHeight=11
CursorType=block
Locale=zh_CN
Charset=UTF-8
Scrollbar=none
Window=max
