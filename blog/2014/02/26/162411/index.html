IBus Chinese Pinyin IM on Linux Mint

Feb 26th, 2014 4:24 pm

Following this:

  1. Install ibus-pinyin:

    $ sudo apt-get install ibus ibus-clutter ibus-gtk ibus-gtk3 ibus-qt4 $ sudo apt-get install ibus-pinyin $ im-switch -s ibus

  2. Restart machine;

  3. Right-click the ibus-pinyin icon in system tray, select “Preference”, then “IBus Preferences” panel open;

  4. On this panel, under tag “Input Method”, check “Customize active input methods”, select “Pinyin”, then click “Add” button;

Now use Ctrl-Space to swtich between IBus-Pinyin and english input method.

The pro of ibus-pinyin is no need to modify system-wide configurations.

2014-7-22:

  1. sudo apt-get install ibus-pinyin ibus-gtk;

  2. ibus-setup, add “PinYin” to the System;

  3. Logout and login;

Bingo.
