Add Xfce to Scientific Linux

Jul 10th, 2014 3:37 pm

  1. Download epel repo rpm from http://fedoraproject.org/wiki/EPEL;

  2. rpm -ivh epel-release-6.8.noarch.rpm;

  3. verify: yum -v repolist, you can see a ‘epel’ in “repo id” column;

  4. yum groupinstall xfce

  5. logout and you can see “xfce seesion” on the bottom of the login screen.

You can still use konsle as default termianl.
