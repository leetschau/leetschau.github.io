方法 ：在_vimrc文件里加上如下代码：

function Dep_to_Install()

“ 将ivy的dependency条目转换为脚本安装条目

%s/dependency/ivy:install/g

%s/org=/organisation=/g

%s/name=/module=/g

%s/rev=/revision=/g

%s/conf=“compile->master”/from=“${from.resolver}” to=“${to.resolver}”/g

endfunction

command Dep2Inst call Dep_to_Install()

map :Dep2Inst
