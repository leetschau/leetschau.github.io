GTD Workflow Based on Emacs

Oct 23rd, 2013 10:28 am

  1. Global Configuration: convert all tasks to appointment, so make it popup automatically: add “(org-agenda-to-appt)” into ~/.emacs;

  2. Create a org file named “jobs.org”;

  3. Add following line to head of the file:

#+SEQ_TODO: TODO(t!) WAIT(w@) | DONE(d!) CANCELED©

  1. Create task: M-S-RET, the input task name: “Notify me”

  2. [optional] Set prioriy: <>;

  3. [optional] Add tag: C-c C-c

  4. Add Schedule: <>, then input date and time, you can specify date and time absolutely or relatively;

  5. Save file: C-x C-s;

Note:

  1. If the appointment is at today, use <> to convert it to appointment manually;

  2. You have many ways to input scheduled date and time, see 8.2.1 The date/time prompt in Org Mode Manual for detail. Or you can read this in Emacs:

M-x info -> Org Mode -> Dates and Times -> Creating timestamps -> The date/time prompt
