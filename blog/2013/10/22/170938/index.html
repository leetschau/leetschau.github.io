Use Reminder in Emacs

Oct 22nd, 2013 5:09 pm

Configuration

Add the following line to ~/.emacs:

(setq appt-issue-message t)

Add Task

M-x appt-add

Cancel Task

M-x appt-delete

Ref: http://dsec.pku.edu.cn/~jinlong/emacs/emacs.html
