Define Key Shortcuts in Emacs

Sep 26th, 2013 4:48 pm

  • Set a keybind: (global-set-key )

“(global-set-key )” and “(define-key global-map )” are basically the same thing in most cases;

  • Find out the current keybinding of a key, for example, C-x C-e: press [F1 k], then press [C-x C-e];

  • List all current key bindings: [F1 b];

  • Get last executed command, for example: use “C-x b” to switch to another window:

  • Press C-x b, then select “Messages”;

  • Press C-x , then you can see the last executed command “(switch-to-buffer ”Message“)”;
