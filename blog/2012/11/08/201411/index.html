Android Hello World程序开发过程

Nov 8th, 2012 8:14 pm

按照Building Your First App，详细过程如下：

  1. 安装SDK（如果网速慢，需要用离线安装的方法，见笔记 离线安装Android SDK的方法 ）；

  2. 采用命令行开发方法（不用装Eclipse插件ADT）；

  3. $SDK_HOME/tools下运行android list targets（$SDK_HOME=C:\Program Files\Android\android-sdk）；

  4. $SDK_HOME/tools下运行"android create project –target 2 –name MyFirstApp –path e:/MyDoc/AndroidEx/MyFirstApp –activity MainActivity –package com.example.myfirstapp"；

  5. 生成一个虚拟设备：$SDK_HOME/tools下运行"android avd"，Name: ChadAVD, Target: Android 4.1.2, CPU/ABI: ARM;

  6. 启动此设备：Android Virtual Device Manager中选中"ChadAVD"，点"start"；

  7. 修改应用界面：e:\MyDoc\AndroidEx\MyFirstApp\res\layout\main.xml修改其中"android:text"的值为"你好安卓！"；

  8. 编译应用：项目根目录（e:/mydoc/androidEx）下运行"ant debug";

  9. 安装应用：E:\MyDoc\AndroidEx\MyFirstApp>“c:\Program Files\Android\android-sdk\platform-tools\adb.exe” install bin/MyFirstApp-debug.apk；

  10. 验证安装成功：在应用列表里可以看到MainActivity，点击，可以看到"你好安卓！"；

  11. 如果修改了源代码，需要在AVD里卸载MainActivity，然后再用adb install安装，否则报“App already exists”错误。
