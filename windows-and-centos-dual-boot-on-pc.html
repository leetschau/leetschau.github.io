Windows and CentOS Dual Boot on PC

  1. Install Windows 7;

  2. Install CentOS 6.4, choose "Basic Storage Devices -> ... -> Create Custom Layout", delete extended partition, create a root partition and a swap partition for CentOS;

  3. Verify the startup list;

  4. After installation of CentOS, login the system, you can edit /boot/grub/menu.lst to modify defaut startup system, default timeout, etc.

  5. During system startup, press up/down key before countdown over to activate the system boot menu to choose Windows or CentOS for startup.

That's it.
