i3 Window Manager on CentOS 6

  1. Make a usb key with dd command and CentOS-6.5-i386-LiveDVD.iso;

  2. Install CentOS 6.5 on Dell PC;

  3. Run firefox and install H3C client to connect to internet;

  4. After installation, run yum update and install epel;

  5. Search i3 in CentOS repository: yum list i3 => i3.i686

  6. yum install i3.i686;

  7. yum install i3lock.i686;

  8. modify /etc/inittab, set the runlevel to 3;

  9. create ~/.xinitrc for user chad: echo "exec i3" > ~/.xinitrc;

  10. reboot;

  11. Login as user chad;

  12. startx;
