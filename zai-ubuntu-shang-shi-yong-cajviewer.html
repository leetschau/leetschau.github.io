在 Ubuntu 上使用 CAJViewer

下载 CAJViewer 安装包 CAJViewer7.2.0.installer.exe 到 ~/Downloads 下， 执行 wine CAJViewer7.2.0.installer.exe，安装完成后，启动： wine ~/.wine/drive_c/Program\ Files\ \(x86\)/TTKN/CAJViewer\ 7.2/CAJVieweru.exe。

CAJViewer 的 打开文件 对话框中不能正确显示汉字文件名 （中文字显示为一个方框，英文和数字显示正常）， 解决方法：执行 winetricks，选择 Select the default wineprefix > Install a font， 在字体列表窗口中选择 fakechinese 和 wenquanyi， 安装完成后，重新启动 CAJ Viewer 后，打开文件对话框能够正常显示中文文件名了。
