Julia Notes

切换模式

]: 包管理模式

;: shell 模式

?: 帮助模式

退格键：返回标准模式

REPL

显示函数签名：输入函数名和左括号后按 TAB；

显示函数文档：进入帮助模式后输入函数名；

DataFrame

  • nrow: row number

  • names: column names

  • maximum(spd.hyd): max of column hyd of dataframe spd

  • argmax(spd.hyd): the index of the maximum of column hyd of dataframe spd

Windows

启动Julia环境

在 Windows 10 的 cmder 启动环境定义文件 user-profile.cmd 中添加： set PATH=%PATH%;%USERPROFILE%\AppData\Local\JuliaPro-0.6.0.1;%USERPROFILE%\AppData\Local\JuliaPro-0.6.0.1\Julia-0.6.0\bin

  • 不需要绘制图形，使用Julia命令行版本：julia 绝对路径：C:\Users\lee_c\AppData\Local\JuliaPro-0.6.0.1\Julia-0.6.0\bin\julia.exe

  • 需要绘制图形：使用IJulia web service：ijulia 绝对路径：*C:\Users\lee_c\AppData\Local\JuliaPro-0.6.0.1\IJulia.bat
