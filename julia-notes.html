Julia Notes

启动Julia环境

在Windows 10的cmder启动环境定义文件 user-profile.cmd 中添加： set PATH=%PATH%;%USERPROFILE%\AppData\Local\JuliaPro-0.6.0.1;%USERPROFILE%\AppData\Local\JuliaPro-0.6.0.1\Julia-0.6.0\bin

  • 不需要绘制图形，使用Julia命令行版本：julia 绝对路径：C:\Users\lee_c\AppData\Local\JuliaPro-0.6.0.1\Julia-0.6.0\bin\julia.exe

  • 需要绘制图形：使用IJulia web service：ijulia 绝对路径：*C:\Users\lee_c\AppData\Local\JuliaPro-0.6.0.1\IJulia.bat
